# TE149-motor-brushless
Controlador PID Digital para motor brushless - Disciplina TE149 - Instrumentação Eletrônica - UFPR 2016/1

##### clone
```bash
$ git clone --recursive https://github.com/wsilverio/TE149-motor-brushless.git
```

##### firmware
```bash
$ make flash
```

##### panel
```bash
$ make panel
$ ./brushless_panel.run
```
